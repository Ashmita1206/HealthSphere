## How to Add Screenshots

1. Run the app locally: `npm run dev`
2. Take screenshots of each page (use browser DevTools to simulate different devices)
3. Save screenshots as PNG files to this directory
4. Update `README.md` to include image references:

```markdown
![Dashboard](./screenshots/dashboard.png)
![Reminders](./screenshots/reminders.png)
![Chat Assistant](./screenshots/chat.png)
```

## Screenshot Dimensions

For consistency, use:
- Width: 1280px (desktop) or 375px (mobile)
- Height: auto
- Format: PNG
- Size: optimized for web (max 500KB each)
